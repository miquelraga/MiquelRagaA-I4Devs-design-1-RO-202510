## Prompt 1 - Diseño conceptual
Actúa como un diseñador de producto experto en sistemas de reclutamiento. Propón el diseño conceptual de la primera versión de un ATS, incluyendo los módulos esenciales (gestión de candidatos, ofertas, pipeline de selección, panel de reclutadores). Describe la experiencia del usuario (UX) y la interfaz (UI) con enfoque en simplicidad y eficiencia. Incluye wireframes conceptuales y flujos de navegación.
***

## ✅ **Módulos esenciales**

1.  **Gestión de candidatos**
    *   Base de datos centralizada con perfiles completos.
    *   Importación desde CV, LinkedIn, o formularios.
    *   Etiquetas y notas colaborativas.
    *   Historial de interacciones.

2.  **Gestión de ofertas**
    *   Creación y publicación de ofertas.
    *   Estado de cada oferta (activa, cerrada, en borrador).
    *   Integración con portales externos (LinkedIn, Indeed).

3.  **Pipeline de selección**
    *   Etapas personalizables (Aplicación → Screening → Entrevista → Oferta → Contratación).
    *   Vista Kanban para arrastrar candidatos entre etapas.
    *   Alertas y recordatorios automáticos.

4.  **Panel de reclutadores**
    *   Dashboard con métricas clave (tiempo medio de contratación, candidatos activos, ofertas abiertas).
    *   Tareas pendientes y próximos pasos.
    *   Colaboración entre miembros del equipo.

***

## ✅ **Experiencia del Usuario (UX)**

*   **Principios clave**:
    *   **Simplicidad**: Menos clics para tareas frecuentes.
    *   **Consistencia**: Navegación clara, iconografía estándar.
    *   **Velocidad**: Acciones rápidas (drag & drop, búsqueda instantánea).
    *   **Colaboración**: Comentarios y menciones dentro del sistema.

*   **Flujos principales**:
    1.  **Crear oferta → Publicar → Recibir candidatos → Gestionar pipeline → Contratar**.
    2.  **Buscar candidato → Añadir notas → Mover en pipeline → Programar entrevista**.

***

## ✅ **Interfaz (UI)**

*   **Estilo visual**:
    *   Minimalista, colores neutros con acentos (ej. azul para acciones).
    *   Tipografía clara (ej. Inter, Roboto).
    *   Botones primarios bien destacados.

*   **Componentes clave**:
    *   **Sidebar**: Navegación principal (Ofertas, Candidatos, Pipeline, Panel).
    *   **Top bar**: Búsqueda global + accesos rápidos.
    *   **Cards**: Para candidatos y ofertas, con info resumida.
    *   **Kanban board**: Para pipeline.

***
