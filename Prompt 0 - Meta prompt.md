# Prompt Inicial para obtención de prompts
Soy un product manager que voy a generar una aplicación de ATS (Applicant-Tracking System) desde cero  Necesito prompts para GPT-5 para las diferentes etapas del diseño del software y las funcionalidades clave
Mi misión es diseñar la primera versión del sistema
Las etapas por las que quiero que me des el prompt más idóneo son:
-Diseño
-Documentación de investigación
-Documentación de análisis
-Casos de uso
-modelado de datos 
-diseño de alto nivel
Separa los prompts por cada undo de los puntos indicados


### ✅ **1. Diseño**

**Prompt:**

> *“Actúa como un diseñador de producto experto en sistemas de reclutamiento. Propón el diseño conceptual de la primera versión de un ATS, incluyendo los módulos esenciales (gestión de candidatos, ofertas, pipeline de selección, panel de reclutadores). Describe la experiencia del usuario (UX) y la interfaz (UI) con enfoque en simplicidad y eficiencia. Incluye wireframes conceptuales y flujos de navegación.”*

***

### ✅ **2. Documentación de Investigación**

**Prompt:**

> *“Genera un documento de investigación sobre el mercado actual de ATS. Incluye análisis de competidores, tendencias tecnológicas (IA, automatización), necesidades de los usuarios (reclutadores y candidatos), y regulaciones clave (GDPR, EEOC). Resume oportunidades y riesgos para un nuevo ATS.”*

***

### ✅ **3. Documentación de Análisis**

**Prompt:**

> *“Crea un análisis detallado para la primera versión de un ATS. Incluye objetivos del producto, alcance funcional, restricciones técnicas, y criterios de éxito. Presenta un mapa de stakeholders y sus necesidades principales. Prioriza funcionalidades usando MoSCoW (Must, Should, Could, Won’t).”*

***

### ✅ **4. Casos de Uso**

**Prompt:**

> *“Genera una lista completa de casos de uso para un ATS básico. Incluye actores (reclutador, candidato, administrador), precondiciones, flujo principal y alternativo, y resultado esperado. Presenta los casos en formato estándar UML.”*

***

### ✅ **5. Modelado de Datos**

**Prompt:**

> *“Diseña el modelo de datos para un ATS que gestione candidatos, ofertas, etapas del proceso y usuarios internos. Incluye entidades, atributos, relaciones (1:N, N:M), claves primarias y foráneas. Sugiere optimizaciones para búsquedas rápidas y filtrado avanzado.”*

***

### ✅ **6. Diseño de Alto Nivel**

**Prompt:**

> *“Propón el diseño de alto nivel para la arquitectura de un ATS escalable. Incluye componentes principales (frontend, backend, base de datos, API Gateway), tecnologías recomendadas, y justificación de las decisiones. Presenta el diagrama de arquitectura y explica cómo se integran los módulos.”*
